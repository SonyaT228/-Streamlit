# 🐱 Cat Health Predictor - Анализ и прогнозирование здоровья кошек

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-link.streamlit.app)

## 📋 О проекте

Веб-приложение для анализа и прогнозирования здоровья кошек на основе данных о породах (Ангорская, Мейн-кун, Рэгдолл). Приложение использует машинное обучение для оценки вероятности здоровья кошки на основе её параметров.

### Возможности:
- 📊 Загрузка и визуализация данных о кошках
- 🔍 Фильтрация по породе, весу, цвету шерсти
- 📈 Интерактивные графики (гистограммы пород, средний вес)
- 🤖 Прогнозирование здоровья с помощью Random Forest
- 🎯 Отображение вероятности предсказания

## 📊 Источник данных

Датасет содержит информацию о кошках трех пород:
- **Ангорская** (Angora)
- **Мейн-кун** (Maine coon)
- **Рэгдолл** (Ragdoll)

Основные признаки: возраст, вес, время игр, продолжительность сна, пол, стерилизация, страна проживания и др.
